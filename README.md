# Data Structure Optimizer

A C++ application with Qt GUI that analyzes datasets and recommends the optimal data structure.

## Project Structure
- `core/` - Data structure implementations (HashMap, BST, Trie, Heap)
- `benchmark/` - Performance testing framework
- `analyzer/` - Recommendation engine
- `gui/` - Qt interface components
- `tests/` - Unit tests
- `docs/` - Documentation

## Build Instructions
```bash
mkdir build
cd build
cmake ..
cmake --build .
./DataStructureOptimizer
```
